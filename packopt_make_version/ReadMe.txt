(1). 当アプリの概要

    実行ファイル自動作成での#packoptのversionキーワードのファイル作成を補助します。



(2). #packoptについて

    #packoptについて詳しくは下記のWebページをご覧ください。

    「プログラミング・マニュアル〜2.9.実行ファイルの作成方法」
    https://www.onionsoft.net/hsp/v37/doclib/hspprog.htm#MAKE_EXE



(3). 使い方

    1. 出力する項目のチェックボックスにチェックを入れてください。

    2. チェックを付けた項目のテキストボックスに値を入力してください。

    3. 「テキスト作成」ボタンをクリックしてください。

    4. 「クリップボードにコピー」をクリックするか、「名前を付けて保存」をクリックしてください。

    5. 終わり。



(4). 「チェックをデフォルトに戻す」ボタンについて

    「チェックをデフォルトに戻す」ボタンをクリックすると、以下のチェックボックスがON状態に戻ります。
    それ以外の項目のチェックボックスはOFFになります。

    ・_FILEVERSION(ファイルバージョン)

    ・_PRODUCTVERSION(製品バージョン)

    ・FileDescription(ファイルの説明)

    ・LegalCopyright(著作権)

    ・OriginalFilename(元のファイル名)

    ・ProductName(製品名)



(5). 履歴

    2022/05/09 (Ver0.2) : 「Versionファイルを開く」ボタンから開かれたファイルの場合、ファイル内にない項目がある場合、その項目のチェックボックスのチェックをOFFにするように修正。
